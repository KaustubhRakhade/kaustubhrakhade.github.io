About Me

I started with Command Prompt as my first programming language, lol. Then I drifted to a more "proper" programming language - Python. I was completely self-taught as our school had no plans on teaching coding anytime in the next 10 years. Then I moved to Android App Development and created some useless Apps mostly for fun. At pretty much the same time, I bought an Arduino and had to learn it's "C"ish language.

But in past few months, I got keenly interested in Web Development. I learned HTML, CSS and Vanilla Javascript and started creating my own Web Apps. At this point, Javascript has surpassed Python as my favourite programming language (I do keep geting JS hangovers and keep breaking my head over why 'list' object has no attribute 'length' ).

I make Web Apps mostly to explore the languages and I'm still in the process. Maybe once I get to college, that I'd be able to dedicate more time to coding. And that's pretty much the story. Thank you for your patience to read on till here and find nothing interesting.
